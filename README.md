1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Fahim-Prodhan/job-task-server.git
   cd YourRepository
  
2. **Install Dependencies**:
   ```bash
    npm install
   
3. **Run**:
   ```bash
    npm run dev
